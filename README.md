## command   
cd 디렉토리변경    
cd . (현재디렉토리) cd .. (상위 디렉토리)    

```
$ cd   
user@:~$   
    
$ cd ./Music   
user@:~/Music  
``` 
   
ls [옵션] 파일 출력    

```
$ ls -l
-rw-rw-r-- 1 user user 3301828 2월 14 18:00 text.txt

$ ls -a
.  sample.mp3 sample2.mp3 sample2.wav 
.. sample.ogg
```

